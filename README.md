# 微信小游戏 - 多人在线飞行棋

这是一个基于原生微信小游戏框架开发的多人在线飞行棋游戏，包含创新的道具系统。

## 项目结构

```
/workspace/
├── game/
│   ├── game.js              # 主游戏逻辑文件
│   ├── game.json            # 微信小游戏配置文件
│   ├── index.html           # 游戏入口HTML文件（用于调试）
│   ├── project.config.json  # 项目配置文件
│   ├── js/
│   │   ├── propSystem.js    # 道具系统
│   │   ├── network.js       # 网络通信模块
│   │   └── gameLogic.js     # 游戏逻辑扩展
│   ├── images/              # 图片资源目录
│   ├── audio/               # 音频资源目录
│   └── libs/                # 第三方库目录
```

## 功能特性

### 基础玩法
- 经典飞行棋规则：掷骰子、移动棋子、撞击对手、到达终点
- 支持2-4名玩家同时游戏
- 流畅的动画效果和用户界面

### 创新道具系统
1. **加速卡** - 让棋子额外移动3步
2. **冰冻卡** - 冻结对手一轮行动
3. **传送卡** - 将棋子传送到任意位置
4. **护盾卡** - 防止被撞回起点
5. **双倍卡** - 下次掷骰子点数翻倍
6. **交换卡** - 与对手交换棋子位置
7. **后退卡** - 让对手棋子后退5步
8. **透视卡** - 查看对手手牌

### 多人在线功能
- 实时网络对战
- WebSocket通信
- 游戏状态同步
- 玩家加入/离开处理

## 游戏规则

1. 玩家轮流掷骰子，根据点数移动棋子
2. 掷到6点可以起飞棋子或额外行动一次
3. 棋子移动到对手棋子位置时，对手棋子返回起点
4. 使用道具卡获得特殊能力
5. 最先让所有棋子到达终点的玩家获胜

## 开发说明

### 核心模块

1. **主游戏逻辑 (game.js)**: 游戏状态管理、渲染、基本玩法
2. **道具系统 (propSystem.js)**: 道具管理、效果应用
3. **网络模块 (network.js)**: 多人在线通信
4. **游戏逻辑扩展 (gameLogic.js)**: 复杂游戏规则和交互

### 微信小游戏API使用

- `wx.getSharedCanvas()` - 获取游戏画布
- `wx.connectSocket()` - WebSocket连接
- `wx.onShow/wx.onHide()` - 游戏生命周期

## 运行方式

1. 在微信开发者工具中打开 `/workspace/game` 目录
2. 或直接在浏览器中打开 `index.html` 进行调试
